Архитектура андроид OS:
-- Linux Kernel

Отвечает за управление памятью, потоками, процессами, драйверами и управление аппаратным обеспечением. На данном уровне реализован механизм Binder, позволяющий взаимодействовать между процессами.

-- Hardware abstract layer

Android предоставляет стандартные интерфейсы для взаимодействия с апаратным обеспечением, а верндоры либо производители железа его реализовывают. Это позволяет системе работать на разном железе, т.к. гуглу не нужно писать драйверы под каждое устройство, а производители могут скрывать исходный код не нарушая правил GPL лицензии. 
Реализации интерфейсов лежат в папке /vendor в формате .so.
